`**uBlock Origin Lite**` - Эффективный блокировщик веб-элементов. Сразу после установки блокирует рекламу, трекеры, майнеры и многое другое.

https://chromewebstore.google.com/detail/adguard-extra/gkeojjjcdcopjkbelgbcpckplegclfeg



`**Privacy Badger**` - Автоматически учится блокировать скрытые трекеры. Сделан ведущей некоммерческой группой EFF, чтобы остановить веб-отслеживание.
